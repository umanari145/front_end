親500px

.child1_2{
 flex: 3 1 250px;
}

.child2_1{
 flex: 1 1 150px;
}

growは余白のコントロール
child1 : 250+(500-(250+150) / (1+3) * 3) = 325px
child2 : 150+(500-(250+150) / (1+3) * 1) = 175px
